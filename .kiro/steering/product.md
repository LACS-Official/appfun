# Product Overview

APPFUN is a modern software management platform built on the LACS API. It provides software browsing, discovery, download management, and activation code handling.

## Core Features

- Software catalog with version history and download links
- Category and tag-based organization system
- Smart search with filtering and sorting
- Responsive design for desktop and mobile
- User authentication via Supabase (optional for downloads)
- Analytics integration with Umami

## Target Audience

Users looking to discover, browse, and download software with a modern, intuitive interface.

## Key Pages

- Home: Featured and latest software
- Categories: Browse by software category
- Tags: Tag cloud and tag-based filtering
- Software Details: Complete software information with versions
- Search: Multi-criteria search with history
- Account: User profile and download management (protected routes)
